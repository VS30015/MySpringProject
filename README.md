# Online Grocery Order Management System

## Description
A Spring Boot REST API to manage Customers, Grocery Items, and Orders for an online grocery ordering system.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database (can be switched to MySQL)
- Maven

## Features
- CRUD operations for Customers
- CRUD operations for Grocery Items
- CRUD operations for Orders
- JPA relationships (OneToMany, ManyToMany)
- Global exception handling
- Embedded Tomcat server

## Run the Application
```bash
mvn spring-boot:run
